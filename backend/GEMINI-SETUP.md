# Groq API Setup (FREE & FAST)

## Get Your Free API Key (1 minute)

1. **Go to Groq Console**: https://console.groq.com/
2. **Sign up** with your email (free account, no credit card needed)
3. **Go to API Keys**: https://console.groq.com/keys
4. **Click "Create API Key"**
5. **Copy the API key**

## Install Package

```bash
cd backend
npm install groq-sdk
```

## Add to Your Project

1. Open `backend/.env` file
2. Add this line:
   ```
   GROQ_API_KEY=your-groq-api-key-here
   ```
3. **Restart your backend server**

## Restart Backend

```bash
npm run start:dev
```

## Troubleshooting

If you get an error:
1. Make sure `GROQ_API_KEY` is in your `.env` file
2. Restart the backend server: `npm run start:dev`
3. Check the API key is valid at https://console.groq.com/keys
4. Make sure you installed the package: `npm install groq-sdk`
